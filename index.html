The six-layer foundation — expand a layer for its certified witness in this repo

1 multidimensional representation — typed axes, not bigger arrays
The integrated state 𝒳[h,n,c,r,u,o,d,m,τ,p,a,t,ρ] is a typed tensor: some axes are states, some are choices, some are constraints, some are generated, some observed, some synchronized, some compressible, some never aggregable. Witnesses: named-axis contraction ≡ Kronecker-lifted operator (tested); grouped-simplex projection 7e-15; 597× sparse-storage measurement.
2 conservation & physical structure
Network flow, cumulative flow, queue propagation, capacity, incidence, activity-time conservation. Witnesses: chain mass conservation 1e-13; fluid-queue pulse ≤ 7e-5 vs closed form (demo 3 above); queue causality on real detector inflow.
3 variational equilibrium & fixed points
UE/SUE, variational inequalities, complementarity, demand–supply closure. Witnesses: entropy-master KKT ≡ logit (exact — demo 1); τ→0 ladder to deterministic UE; mixed-autonomy closure with final-state consistency residuals.
4 numerical optimization
Frank–Wolfe, gradient projection, ADMM, column generation, major/minor decomposition — the solver layer, chosen after state, operators, and conservation are fixed. Witnesses: Chicago 93,135 ODs @ 9.8e-5 in 12 s; TRMG2 1,039,117 ODs @ 4.4e-7 (demo 2 is this machinery in miniature); the engine-escalation ladder with its measured stiff-block lesson.
5 compression & scalable computation
Sparse tensors, CP/Tucker/TT, generated columns, low rank, regional decomposition. Witnesses: the rank-economy sweep (bias 2e-5 → 2.95 as K: 2→16; promotions sublinear); the K̄ ≈ 2.2 compression-admissibility boundary — reported honestly, including where compression cannot pay.
6 modern neural computation
Computational graphs, autodiff, adjoints, PINN, neural operators, constrained attention Yi = Σj Kij(𝒵) Xj with masks from incidence, causality, and availability. Witnesses: softmax router ≡ logit (bitwise); hand adjoint ≡ autograd 1e-9; bounded residual admissibility; the learned-proposer honest negative. These layers must connect — learning only this one is the failure mode the framework is designed against.
