TensorMobility

Mobility systems as flow-through tensors — space, time, and behavior in one certified computational graph. The software home of the STB-FTT research line (asu-trans-ai-lab).

The unified equation

𝒵★ = ΠΩ [ 𝒦struct(𝒵★, 𝒰) + 𝒩θ(𝒵★, 𝒰) ]

multi-axis urban state 𝒵 = {ℒ, ℋ, 𝒜, 𝒟, ℱ, 𝒮, 𝒴} · structural land-use–ABM–DTA–queue–observation operators 𝒦 · trainable operators 𝒩θ (PINN / neural operator / residual) · projection ΠΩ onto conservation, capacity, temporal, and behavioral feasibility.

Neural networks are not the starting point; tensor decomposition is not the starting point; ADMM is not the starting point. The starting point is: which states compose the urban system, and through which conservation operators are they connected? The chain is a sequence of typed operators, never a file hand-off:

𝒜 → demand formation → 𝒟 → path generation → ℱ → dynamic network loading → 𝒮 → experienced costs → 𝒜+

The six-layer foundation — each layer proven by a certified case

1 multidimensional representation named-axis contraction ≡ Kronecker-lifted operator (tested); grouped-simplex projection 7e-15; 597× sparse-storage measurement
2 conservation & physical structure chain mass conservation 1e-13; fluid-queue pulse ≤ 7e-5 vs closed form; queue causality on real detector inflow (IEEE corridor)
3 variational equilibrium & fixed points entropy-master KKT ≡ logit (exact); τ→0 ladder to deterministic UE; mixed-autonomy closure with consistency residuals
4 numerical optimization full-space certified column generation: Chicago 93,135 ODs @ 9.8e-5 in 12 s; TRMG2 1,039,117 ODs @ 4.4e-7; engine-escalation ladder with the measured stiff-block lesson
5 compression & scalable computation rank-economy sweep (bias 2e-5 → 2.95 as K: 2→16; promotions sublinear); the K̄ ≈ 2.2 compression-admissibility boundary, reported honestly
6 modern neural computation softmax router ≡ logit (bitwise); hand adjoint ≡ autograd 1e-9; bounded residual admissibility; the learned-proposer honest negative

These layers must connect; learning only the last one is exactly the failure mode this framework is designed against. Attention enters as a transportation-constrained, state-dependent kernel Yi = Σj Kij(𝒵) Xj — its masks come from incidence, causality, and availability, never from hyperparameters. A learned loading surrogate is never free-standing: 𝒮 = ΠΩ[ΦDNL(𝒟) + rθ(𝒟, 𝒴)], never 𝒮 = NN(𝒟).
